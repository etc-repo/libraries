# 라이브러리 정리방법
- markdown으로 깃헙 공동 repo에서 정리
- 주로 쓰이거나 필요할 것 같은 클래스 또는 문법 5개 정리해오기
- 정리 대상: `클래스의 설명과 사용법`, `메소드 설명`, `예시 코드` 기타 등 자유롭게
- 너무 자세하게 정리할 필요 없음 (클래스에서도 필요할 것같은 메소드만 정리하기) (실용측면)
- 정리할 것(예. 어떤 클래스) 선택 자유 (겹쳐도 상관없음)
